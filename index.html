Chosen Career: Network Security Analyst

A Network Security Analyst monitors network traffic, detects security threats, analyzes suspicious activities, and helps protect an organization’s network from cyber attacks.

Job Responsibilities

  • Monitor network traffic and security alerts
  • Detect unauthorized access and cyber attacks
  • Analyze packets using Wireshark
  • Prepare security reports and incident findings

Required Skills

  • Networking fundamentals
  • Cybersecurity knowledge
  • Wireshark packet analysis
  • Linux basic command skills
  • Problem-solving and communication
