Career Exploration

Chosen Career: Network Security Analyst

A Network Security Analyst is responsible for monitoring network traffic, detecting security threats, analyzing suspicious activities, and protecting an organization’s network from cyber attacks.

Job Responsibilities

  • Monitor network traffic and security alerts.
  • Detect unauthorized access and cyber attacks.
  • Analyze packets using tools such as Wireshark.
  • Prepare security reports and incident findings.
  • Support network protection and security improvement.

Required Skills

  • Networking fundamentals
  • Cybersecurity knowledge
  • Packet analysis using Wireshark
  • Linux basic command skills
  • Problem-solving and communication skills

Relevance to Cybersecurity

This role is important because modern organizations depend on secure networks. Network Security Analysts help prevent data breaches, detect attacks, and ensure network availability for users.
